#  Панель моніторингу: Котельня 

![Preview](/LAB2/img/Screenshot.png)

## Опис проєкту
Цей проєкт є вебдодатком для моніторингу показників енергетичного об'єкта (Котельні) в режимі реального часу. Розроблено в рамках практичної роботи №2 з дисципліни "Основи веб-програмування".

##  Технології
* **HTML5:** Семантична розмітка сторінки.
* **CSS3:** Стилізація з використанням CSS Grid, Flexbox, медіа-запитів для адаптивності та CSS-змінних.
* **JavaScript (Vanilla JS):** Логіка генерації випадкових даних, оновлення DOM-елементів та керування таймерами.

##  Основний функціонал
* Відображення 8 ключових параметрів котельні (температура, тиск, витрата газу тощо).
* Динамічне оновлення даних за допомогою JavaScript.
* Кнопка ручного оновлення даних.
* Кнопка автоматичного оновлення (таймер кожні 3 секунди).
* Кольорова індикація статусів (норма, попередження, критично) з анімацією для критичних значень.

##  Як запустити
1. Відкрийте файл `index.html` у будь-якому сучасному веб-браузері (Chrome, Firefox, Safari, Edge). Додатковий сервер не потрібен.
